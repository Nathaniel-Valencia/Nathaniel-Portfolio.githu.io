Projects

LAC Dashboad - MVP

Created Google Spreadsheet Dashboards in order to help the company track employee's and team's performance.

The Dashboard includes "Attendance" (to determine if an employee is Active on site or on leave), "Total Completion Rate" (to measure what percentage of the target is being completed by the employee), "Total Utilization Rate" (to determine if the employee is under/over utilized at work), and Total Output produced

A breakdown of output is also included on the Dashboard to give an overview on what has already been processed and on which functions the processed job belongs to.

Counters

Multiple counters have been created in order to produce the desired dashboard output.

The cells on each counter contains formula in order to sum up the total number of jobs processed by an employee on a given task, on a given day. The formula used on this project are mostly SUMIFS and COUNTIFS formula, and here's an example:

Formula: SUM(COUNTIFS('Build WAPs'!$A:$A,Dashboard!$A$1,'Build WAPs'!$B:$B,$A9,'Build WAPs'!$C:$C,"*Completed*",'Build WAPs'!$F:$F,"*- 2 -*"),COUNTIFS('Build WAPs'!$A:$A,Dashboard!$A$1,'Build WAPs'!$B:$B,$A9,'Build WAPs'!$C:$C,"*WIP*",'Build WAPs'!$F:$F,"*- 2 -*"),COUNTIFS('Build WAPs'!$A:$A,Dashboard!$A$1,'Build WAPs'!$B:$B,$A9,'Build WAPs'!$C:$C,"*Validation*",'Build WAPs'!$F:$F,"*- 2 -*"))

Tasks Tabs

The data being processed by the Dashboard is being updated by the employees in real time. Employees are given access to the Dashboard in order to update each job upon completion, updating the Remarks Column into Completed or WIP or Validation then selecting a remarks on the drop down in order to track what's the update and how long it took to complete the job.

Data Validation Tab

The Data Validation Tab deserve it's own section since this contains a lot of the information that makes up the Dashboard. This contains the list of Employee Names, the Statuses, and Remarks that will be used in order to track the the progress of each job and the employee's performance.

And lastly, the data that we collected on the dashboard are being stored, analyze, and visualize in order to determined the Team's Overal Performance on a monthly and annual basis giving the managers an easier time in collating ang analyzing the current status of the company.
