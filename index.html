Introduction

Tracking the Team's and Employee's individual performance is a very important tasks since a lot of the business decisions rely on how each individual function performs within the organization. Having an overview (in the form of data and visuals) of the individual and team performances makes it easier to identify gaps, challenges, resource staffing issue, and possible resolution to an arising problem.

Planning

In the planning stage, I had to set up meetings with my co-team leads in order to know how each functions works. We also discussed the metrics and the parameters that are needed in order to measure employee productivity. We also determined the Actual Handling Time (AHT) for each tasks within each function to know how many jobs each employee can complete within their working hours. With the metrics and parameters, together with the AHT provided, and knowing how to measure each function, we are now ready to start building up the Dashboards

The Dashboard Tab

The Dashboard tab displays summary of how the individual employees are performing on a given day.
Attendance --> to determine if an employee is Active on site or on leave
Total Completion Rate --> to measure what percentage of the target is being completed by the employee
Total Utilization Rate --> to determine if the employee is under/over utilized at work
Total Output --> to keep track of the number of output produced by each employee
Breakdown of Output --> to give an overview on what has already been processed and on which functions the processed job belongs to

Tab Counters

In order to build the Dashboard, we needed to create multiple counters in order to produce the desired output as required by each functions. The counters will calculate the data to be displaying on the Dashboard tab.

Tasks Tabs

The data being processed by the Dashboard is being updated by the employees in real time. Employees are given access to the Dashboard in order to update each job. Upon completion, the Status Column will be updated into Completed or WIP or Validation then selecting a Remarks on the drop down in order to track the job's current status and how long it took to complete the job.

Data Validation Tab

The Data Validation Tab deserve it's own section since this contains a lot of the information that makes up the Dashboard. This contains the list of Employee Names, the Statuses, and Remarks that will be used in order to track the the progress of each job and the employee's performance.

And lastly, the data that we collected on the dashboard are being stored, analyze, and visualize in order to determined the Team's Overal Performance on a monthly and annual basis giving the managers an easier time in collating ang analyzing the current status of the company.
